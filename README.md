# StockMorning

StockMorning 是一个基于 Python 的股票量化交易系统，支持技术指标分析、股票筛选和持仓管理等功能。

## 功能特点

- 多维度技术指标分析（MA、MACD、RSI、KDJ等12个指标）
- 基于多线程的高效数据获取
- 自动股票筛选和信号生成
- 持仓管理和风险监控
- 指数表现跟踪分析

## 系统架构

StockMorning/
├── begin.py # 主要策略实现
├── calculate.py # 指数计算和分析
├── end.py # 持仓管理系统
└── config.yaml # 配置文件
